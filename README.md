### Bazaarvoice's Swift SDK

## Requirements

CocoaPods > 1.5, Xcode, and iOS 8.0

## Installation

BVSwift is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'BSwift'
```
